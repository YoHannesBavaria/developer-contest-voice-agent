# Demo-Call Script (Recording Guide)

## Ziel
Ein kompletter 2-4 Minuten Demo-Call, in dem der Agent qualifiziert und einen Termin bucht.

## Szene
- Lead: Head of Sales, 120 Mitarbeitende, hohe Dringlichkeit.
- Problem: Zu viele manuelle Erstgespraeche, schlechte Erreichbarkeit ausserhalb Buerozeiten.

## Ablauf
1. Begruessung
- Agent erklaert kurz den Zweck und fragt nach Erlaubnis fuer 4 Qualifizierungsfragen.

2. Qualifizierung
- Interesse/Dringlichkeit
- Budget
- Unternehmensgroesse
- Timeline
- Entscheidungsbefugnis

3. Value Framing
- Agent nennt 1-2 konkrete Mehrwerte fuer den genannten Pain Point.

4. Terminbuchung
- Agent bietet zwei Slots an und bestaetigt den gewaehlten Slot.

5. Abschluss
- Agent fasst zusammen (Lead Score, naechste Schritte).

## Aufnahme-Tipp
- Erst einen trockenen Test ohne Aufnahme.
- Dann final mit sauberer Audioqualitaet aufnehmen (Loom oder Screen+Audio).
